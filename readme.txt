/*
* Pre requsite: Please install latest node version.
*/

1. Clone git repository from https://github.com/g9jadhav/assignment.git
2. After cloning repository go to assignment > ToDoList_Assignment directory.
3. Open the same path into the terminal.
4. Run npm install.
5. After installing dependencies.
6. Run command: node server
7. After running above command Open https://localhost:7878 in your browser.
8. Application will get open.
9. Right now default credentials given in email and passowrd field for login.
10. After login your app is ready to test.
